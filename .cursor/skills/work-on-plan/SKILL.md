# Work on plan

Skill for making it more convenient to continue working on a plan.

## When to Apply

Use this skill when:

- You are asked to continue working on an unspecific plan.

## Instructions

- Check the plan in [AGENTS.md](../../../AGENTS.md) and keep working on it. Follow the instructions in the plan.
